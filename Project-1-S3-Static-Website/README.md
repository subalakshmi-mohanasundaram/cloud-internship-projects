# Project 1: Static Website Hosting on AWS S3

## 📌 Description
This project demonstrates how to host a static website using AWS S3.

## 🚀 Steps Performed
- Created an S3 bucket
- Uploaded index.html file
- Enabled static website hosting
- Configured bucket policy for public access
- Accessed website using S3 endpoint URL

## 🌐 Output
Successfully hosted a static website on AWS S3.

## 🛠️ Technologies Used
- AWS S3
- HTML
