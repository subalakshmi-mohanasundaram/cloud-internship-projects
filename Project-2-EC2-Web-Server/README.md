# Project 2: EC2 Web Server Deployment

## 📌 Description
This project demonstrates how to launch a virtual server using AWS EC2 and host a static website using Apache Web Server.

## 🚀 Steps Performed
- Created an EC2 instance (Ubuntu)
- Connected using EC2 Instance Connect
- Installed Apache Web Server
- Navigated to /var/www/html
- Removed default index.html
- Created a custom index.html
- Hosted website using public IP

## 🌐 Output
Successfully deployed a static website on AWS EC2 instance and accessed it via public IP address.

## 🛠️ Technologies Used
- AWS EC2
- Ubuntu Linux
- Apache Web Server
- HTML
